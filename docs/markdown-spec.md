# Markdown 语法规范 v2.0

> DrawRu v3.7.0 · 支持扩展图片语法

## 章节标题

### 主标题

```
# 软件名称
```

### 章节标题（显式编号）

```
## 01 软件介绍
## 02 功能特点
## 03 使用说明
```

编号必须为 `01`-`09` 两位数字。

### 章节标题（自动编号）

```
## 软件介绍       → 自动编号 01
## 功能特点       → 自动编号 02
```

### 子标题

```
### 基本使用
### 高级设置
```

## 信息列表

```
软件名称：TrafficMonitor
版本：V1.86
平台：Windows
```

冒号分隔，标签 1-6 个字符，支持 `：`(全角) 和 `:`(半角)。

## 图片

### 标准语法

```
![截图](https://example.com/202210290944229.jpg)
```

### 扩展属性（Phase 2-B1）

```
![截图](https://example.com/202210290944229.jpg)
{width=800 height=600 align=center border=true caption="系统架构图"}
```

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `width` | 整数 | null | 图片宽度 px |
| `height` | 整数 | null | 图片高度 px |
| `align` | left/center/right | center | 对齐方式 |
| `border` | true/false | true | 是否显示主题边框 |
| `caption` | 字符串 | "" | 图片说明文字 |

### Asset 协议（Phase 2-C.2）

```
![截图](asset:img001)
```

通过 `assetId` 从 ImageAssetManager 解析实际 URL。

## 完整模板示例

```
# TrafficMonitor

## 01 软件概览

软件名称：TrafficMonitor
版本：V1.86
平台：Windows

![截图](https://example.com/202210290944229.jpg)
{width=600 caption="运行界面"}

## 02 主要功能

1. 实时监控网络速度
2. 显示 CPU 和内存使用率
3. 支持任务栏嵌入

> 轻量级系统监控工具，无需安装。

## 03 使用说明

### 基本使用

下载后直接运行即可。

### 高级设置

在设置中可以调整显示位置和颜色主题。
```

## 解析约束

- 图片语法独占一行
- 属性块紧随图片下一行
- 行内图片（段落中嵌入 `![img](url)`）不支持
- 引用式图片（`![alt][ref]`）不支持
- HTML `<img>` 标签会当作文本

## 旧格式兼容

v3.5.5-beta 仍支持以下旧格式（内部转换为 Markdown）：

```
01
【软件概览】
```
**不建议继续使用旧格式**，推荐使用 `## 01 软件概览`。
