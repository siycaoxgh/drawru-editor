【软件概览】

软件名称： TrafficMonitor

软件大小： 约 2MB - 5MB（含 Lite 版与常规版）

软件版本： V1.86

更新日期： 2026/03/29

适配平台： Windows

02


【软件介绍】

TrafficMonitor 是一款运行在 Windows 平台上的极简开源实时监控工具。

它主要用于在桌面上、任务栏中实时显示网速（上传/下载）、CPU 利用率、内存占用以及硬件温度等关键系统信息。

凭借其轻量化的设计、强大的皮肤定制功能以及对多显示器环境的完美支持，它已成为 Windows 用户优化桌面体验、监控硬件状态的装机必备神器。

03


【主要功能】

  1. 多维度实时监控： 实时追踪上传/下载速度、CPU 与内存占用率。在 V1.86 Lite 版中，进一步新增了 GPU（显卡） 和 硬盘利用率 的显示。

  2. 任务栏深度集成： 支持将监控数据嵌入任务栏，并能根据 Windows 主题色自动调整背景颜色。

  3. 多显示器优化： 最新版本支持在副显示器上独立显示任务栏窗口，并解决了多显示器 DPI 不一致导致的界面错位问题。

  4. 高度自定义皮肤： 用户可更换皮肤，V1.86 现已支持为每个皮肤单独保存字体、颜色及显示文本内容。

  5. 多语言本地化： 采用全新的外部 .ini 配置文件重构，新增了德语、意语、俄语、波兰语、土耳其语、希伯来语等多国语言支持。

  6. 智能恢复机制： 优化了系统从睡眠状态恢复后的网络连接重连逻辑，确保网速显示不中断。

04


【使用演示】

  1. 启动与显示： 下载解压后运行主程序，监控悬浮窗将默认出现在桌面右下角。

  2. 设置任务栏模式： 右键点击悬浮窗，在菜单中选择 “显示任务栏窗口”，数据即可嵌入到系统任务栏中，节省桌面空间。

  3. 外观调整：通过右键菜单进入 “选项设置”，可动态调整窗口宽度、字体大小。

  4. 在 “更换皮肤” 对话框中，可通过新增的 ListBox 列表快速预览并切换心仪的样式。

  5. 副屏配置： 若有双显示器，可在设置中开启“在副显示器显示”功能，让主副屏都能随时掌握硬件状态。

  6. 故障排查： 若软件意外崩溃，V1.86 新增的崩溃日志与调用堆栈显示功能，可帮助你将错误信息记录并反馈给开发者。
